# 💻 Intest

## Project Overview

Intest is an online examination platform available on both web and mobile, covering Student, Teacher, and Admin roles. During my internship as a Quality Assurance Intern at PT Humma Teknologi Indonesia, I acted as the sole QA during the early testing phase, responsible for designing test cases and verifying application functionality before being reassigned to another project.

---

## My Role

**Quality Assurance Intern - Solo QA**
**Duration: July 21, 2025 – September 23, 2025**
**Platform: Web, Mobile**
**Roles Tested: Admin, Teacher, Student**
