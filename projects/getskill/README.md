# 📚 GetSkill

## Project Overview

GetSkill is a Learning Management System (LMS) designed to support online learning and training activities. During my internship as a Quality Assurance Intern at PT Humma Teknologi Indonesia, I participated in testing various application features to ensure they met functional requirements before release.

---

## My Role

**Quality Assurance Intern**

---

## Responsibilities

- Created manual test cases based on feature requirements.
- Executed functional testing on application features.
- Performed positive and negative testing.
- Validated application workflows and form validations.
- Collaborated with developers during the testing process.
- Conducted regression testing after fixes.
