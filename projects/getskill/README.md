# 📚 GetSkill

## Project Overview

GetSkill is a Learning Management System (LMS) built to support online learning and training activities, with a multi-role structure covering Admin, Siswa, Author, Mentor, and Sekolah — including online courses, exams, industry classes, and payment transactions. As a Quality Assurance Intern at PT Humma Teknologi Indonesia, I was responsible for designing and executing test cases across the platform's functional, UI, security, and regression aspects, including the Anti-Cheat System and authorization testing via Postman, to ensure application stability and security before every production deployment.
