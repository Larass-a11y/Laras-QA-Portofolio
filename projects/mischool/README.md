# 🏫 MiSchool

## Project Overview

MiSchool is a web-based School Management System covering online learning, exams, and school administration for Teacher, School, and Student roles. During my internship as a Quality Assurance Intern at PT Humma Teknologi Indonesia, I acted as the solo QA on this project, responsible for designing and executing test cases to ensure the application functioned according to business requirements.

---

## My Role

**Quality Assurance Intern - Solo QA**
**Duration: December 26, 2025 – April 1, 2026**
**Platform: Web**
**Roles Tested: Teacher, School, Student**

---

## Responsibilities

- Designed and executed 232 test cases as the sole QA on the project.
- Executed functional, UI, end-to-end, and security testing on application features.
- Performed positive and negative testing.
- Tested the Anti-Cheat System (fullscreen, block right-click, block copy, tab switching detection, time-up handling).
- Verified the ID-card-based attendance feature directly.
- Verified application workflows across all tested roles.
- Collaborated with developers during the testing process.
- Conducted regression testing after fixes.

---

## Features Tested

- E-Learning & Class Management
- Question Bank & Add Question (Multiple Choice, Essay, Matching, Multiple Answer)

- Anti-Cheat System (fullscreen, block right-click, block copy, tab switching, time-up handling)
- ID Card-Based Attendance
- Authentication (Login)
- Dashboard & Score Recap (Teacher, School, Student)
